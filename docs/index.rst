Welcome to mpd-box's documentation!
###################################

MPD-Box start a song in Music Player Daemon by simply scanning a NFC.

It creates a new way to interact with your music. 
Once setup for the first time, just grab a tag and start listening to it without having to navigate a screen interface. Remove it, and it will stop.
